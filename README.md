# Git & GitHub Mastery

A production-ready VitePress course website that teaches Git from beginner to advanced through real-world decision-making, workflow design, recovery techniques, GitHub collaboration, and hands-on labs.

## Live site

https://git-mastery-course.vercel.app

## Author

Sourabh Patware

## Stack

- VitePress
- Markdown content under `course/`

## Scripts

```bash
nvm use 24
npm install
npm run dev
npm run build
npm run preview
```

## Requirements

- Node.js 24
- npm 10 or newer recommended

## Project structure

```text

.vitepress/
  config.ts
  theme/
course/
  git/
  github/
  workflows/
  labs/
  public/
  index.md

```
